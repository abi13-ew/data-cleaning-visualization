# 📊 Netflix Data Analysis Project

## 🔹 Overview
This project focuses on cleaning, analyzing, and visualizing Netflix dataset to extract meaningful insights.

## 🔹 Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## 🔹 Key Insights
- Majority of content is Movies over TV Shows
- USA produces the highest number of shows
- Content growth increased rapidly after 2015
